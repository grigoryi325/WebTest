Создать мини приложение для складского учета. Интерфейс программы должен позволять: 

1.	Отображать в табличном виде реестр продуктов. Поля реестра «Товары»: Название, Количество, Закупочная цена,  Категория товара.
2.	Возможность добавлять новый товар. По кнопке «Добавить» должна отображаться карточка редактирования, содержащая следующие поля: Название(Строка), Количество(Дробное), Закупочная цена(Дробное), Категория(Справочник в виде выпадающего списка [Товар, Услуга])
3.	Провести индексацию цен по всем товарам: Но нажатию на кнопку «Индексация» появляется форма с одним полем «% индексации»(Дробное). При нажатии «ОК» этой формы по всех продуктах происходит пересчет поля «Закупочная цена». «Закупочная цена» = «Закупочная цена»  * («% индексации» + 1). Поле «% индексации» не сохраняется в БД.
4.	Удалять товар из реестра.
